# 🎓 EduSimplify AI

EduSimplify AI is an AI-powered educational platform that transforms complex academic content into personalized learning experiences for different learner levels. Built using IBM Watsonx.ai and IBM Granite, the platform simplifies technical concepts and provides interactive learning tools such as quizzes, flashcards, revision notes, learning roadmaps, and real-world applications.

---


## 🎯 Problem Statement

### IBM SkillsBuild Internship Project

**Problem Statement 19: Course Content Simplification Agent**

Educational materials vary in complexity and are often difficult for learners with different backgrounds.

This project uses Generative AI to analyze academic content and adapt explanations based on learner proficiency, enabling personalized education delivery at scale.

---


## 🚀 Project Overview

Educational content is often difficult to understand because learners have different levels of prior knowledge.

EduSimplify AI solves this problem by:

- Simplifying academic content for different learner levels.
- Generating concise summaries and revision notes.
- Creating interactive quizzes and flashcards.
- Providing learning roadmaps and prerequisites.
- Showing real-world applications of concepts.
- Comparing the same concept across multiple learning levels.

The platform is powered by IBM Granite foundation models hosted on IBM Watsonx.ai.

## 🌐 Application Workflow

1. User enters academic content.
2. User selects learner level.
3. IBM Granite processes the content.
4. Simplified explanation and summary are generated.
5. Additional AI features are available:
   - Revision Notes
   - Flashcards
   - Quiz
   - Learning Roadmap
   - Real-World Applications
   - Compare Levels
